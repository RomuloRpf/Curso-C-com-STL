Problema B

Broken Keyboard (a.k.a. Beiju Text)

Você está digitando um texto longo com o teclado quebrado. Bem... não tão quebrado assim. O único problema com o teclado é que às vezes as teclas "home" ou "end" são automaticamente pressionadas (internamente).

Como você está focado na digitação - e nem ligou o monitor! - esse problema passou despercebido. Depois de terminado o texto, você pôde ver como ele ficou na tela (após ligar o monitor, claro).

Em Chinês, esse tipo de texto é chamado Beiju. Sua tarefa será determinar o texto Beiju.

Input

Haverá vários casos de teste. Cada caso de teste é composto por uma única linha contendo pelo menos uma, e no máximo 100.000 letras, underlines e dois caracteres especiais '[' e ']'. '[' significa que a tecla "Home" foi pressionada internamente, e ']' significa que a tecla "End" foi pressionada internamente. O input é terminado com o fim de arquivo (EOF). O tamanho do arquivo de input não excede 5MB.

Output

Para cada caso, imprima o texto Beiju resultante na tela.

Input de Exemplo

This_is_a_[Beiju]_text
[[]][][]Happy_Birthday_to_Tsinghua_University

Output correspondente

BeijuThis_is_a__text
Happy_Birthday_to_Tsinghua_University
Rujia Liu's Present 3: A Data Structure Contest Celebrating the 100th Anniversary of Tsinghua University
