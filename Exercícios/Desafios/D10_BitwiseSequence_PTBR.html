Problem D: Bit-wise Sequence

Dr. DoLots é um expert em teoria dos números e ele está atualmente estudando números primos. Na verdade, algumas pessoas dizem que ele tem algum tipo de fórmula que é capaz de gerar os números primos em ordem crescente. Entretanto, a fórmula precisa ordenar os números inteiros não negativos de acordo com o crescente número de uns em suas representações binárias. Isto é, todos os inteiros não negativos com m uns em suas representações binárias devem ser ordenados antes de todos os inteiros não-negativos com m+1 uns.

Mais formalmente, vamos definir a sequência de inteiros não-negativos Bn tal que para todo par de números inteiros não negativos m e n onde n<m, um dos dois ítens abaixo são válidos: 
            1. O(Bn)< O(Bm)
            2. Ou O(Bn) = O(Bm) e Bn< Bm.
onde O(n) é o número de uns na representação binária de n.

Dr. DoLots gostaria que você fizesse um programa que computa o valor de Bn para valores arbitrários de n, 0 ≤ n ≤ 2147483647. Para maior simplicidade, nós vamos considerar que apenas inteiros não negativos menores que 2147483648 aparecerão nessa sequência. Você deve estar se perguntando por que o doutor não consegue fazer isso sozinho. Ele está muito ocupado e quer utilizar seu tempo em coisas mais importantes (É isso que ele diz, mas na verdade ele é bem preguiçoso... Mas você pode ajudá-lo, não pode?).

Input

O Input consiste em várias linhas, cada uma especificando um valor inteiro não-negativo para n.

Output

O Output deve seguir o mesmo formato que o input exceto que n é substituído por Bn.

Input de Exemplo

 
0
1
2
31
32

Output Correspondente

 
0
1
2
1073741824
3
